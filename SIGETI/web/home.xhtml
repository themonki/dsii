Bienvenido al Sistema de Gestión de Transporte Integrado (SIGETI)

para el apoyo de los Sistemas de Transporte Masivo. Se cuenta con los siguientes Módulos:



    Funciones del Sistema:

  • Gestionar usuarios: El sistema ofrecerá el sevicio de gestión de usuarios, manejando así el ingreso de nuevos usuarios del STM -pasajeros- (quienes lo harán cuando adquieran una tarjeta personalizada), también el ingreso de nuevos empleados al sistema, este ingreso se hará por parte del gerente del sistema. Esta gestión de usuarios permite modificar los datos ingresados en el sistema tanto de pasajeros como de empleados a excepción de la cédula en ambos casos, finalmente permite la eliminación de un empleado cuando se requiera.
  • Gestionar roles: Para los empleados del sistema que usarán el sistema, este debe diferenciar entre los diferentes cargos (roles) de los empleados, por lo que una vez se ingrese al sistema este identifica y carga las opciones que tiene cada rol para ejercer la función que se le ha asignado en el STM.
  • Administrar rutas: El sistema debe permitir que se ingresen rutas, especificando el nombre, descripción del recorrido y paradas que realiza, como a su vez modificar la información referente a una ruta, esto por parte del director del STM, finalmente permite eliminar una ruta cuando así se decida.
  • Administrar buses: El sistema guardará información de los buses que se ingresen, también a cada bus le asociará una ruta, y debe permitir que se eliminen buses cuando sea necesario.
  • Control y seguimiento: El sistema soporta un modulo para realizar el control y seguimiento de quejas y/o reclamos que realicen los usuarios del STM -pasajeros-.
  • Gestionar tarjetas: En este modulo el sistema debe permitir el ingreso de tarjetas genéricas y personalizadas, el recaudo de ventas y recargas, manejar la información de pasajes y la eliminación de tarjetas cuando así se requiera.
